# Admin UI Guide

Guide to the Mind Measure admin portals — both the University and Pro (Corporate) superuser dashboards.

**Last Updated**: 28 February 2026

---

## Portals

| Portal | URL | Repo |
|---|---|---|
| University | [admin.mindmeasure.co.uk/superuser](https://admin.mindmeasure.co.uk/superuser) | mind-measure-core |
| Pro (Corporate) | [admin.mindmeasurepro.com/superuser](https://admin.mindmeasurepro.com/superuser) | mind-measure-pro-core |

Both portals use the same architecture: React + Vite + TypeScript, deployed on Vercel. The Pro portal mirrors the University portal with corporate branding and company-specific terminology (companies instead of universities, departments/offices instead of faculties).

## Authentication

Superuser access uses a **passwordless 6-digit code flow**:

1. Enter a `@mindmeasure.co.uk` email address
2. Receive a 6-digit code via email (AWS SES)
3. Enter code to authenticate
4. Session persists via JWT token

Only registered Mind Measure staff emails are accepted.

## Superuser Tabs

Both portals share the same tab structure. Superadmin users see all tabs. Non-superadmin users see a subset based on their `user_permission[]` array.

### Overview

The main dashboard with live platform metrics:

- **Network Score**: Aggregate wellbeing score across all institutions
- **Active Users / Check-ins**: Platform engagement
- **Metric Cards**: Score, users, check-ins, engagement, risk, participation
- **Network Trend Chart**: Wellbeing trends over time with selectable range
- **Score Distribution**: Histogram of wellbeing scores
- **Institution Cards**: Quick access to each university/company dashboard
- **Themes and Concerns**: Network-wide positive themes and concerns
- **Security Summary**: Current security posture
- **Priority Alerts**: Items requiring attention
- **Institution Rankings**: Ranked by engagement or wellbeing score
- **Recent Activity**: Latest platform events

### Universities / Companies

Manage institutions:

- **Institution List**: All onboarded universities or companies with status, student/employee count, engagement metrics
- **Add New**: University/company onboarding wizard
- **Dashboard Access**: Click through to institution-specific dashboards
- **Delete**: Remove institutions (with confirmation)

### Users

Superadmin only. Manage Mind Measure staff accounts:

- **User List**: All `core_users` with roles and permissions
- **Permission Management**: Assign `user_permission[]` enum values
- **Invite Users**: Add new staff members

### Marketing / Content Studio

Opens the Marketing CMS ([marketing.mindmeasure.co.uk](https://marketing.mindmeasure.co.uk)) in a new tab with auth token passthrough. Not a tab within the portal — redirects to the external CMS.

### Security

Security monitoring dashboard:

- **Compliance Status**: HIPAA, GDPR, SOC2 tracking
- **Security Score**: Overall security posture
- **Audit Logs**: Real-time security event monitoring
- **Vulnerability Scanning**: Automated security checks

### Costs

AI service cost monitoring:

- **Real-time Spend**: Bedrock, Rekognition, ElevenLabs costs
- **Budget Alerts**: Threshold notifications
- **Usage Analytics**: Per-service utilisation
- **Cost Optimisation**: Recommendations

### System

System health and management:

- **Ecosystem Health Audit**: Runs `/api/system/ecosystem-audit` covering database, APIs, security, deployment, voice agent, mobile app health
- **Platform Releases**: Version history
- **Service Status**: Real-time component health

### Resources

Superadmin only. National mental health resources management:

- **Resource List**: UK-wide mental health services
- **Add/Edit**: Manage crisis lines, support organisations
- **Categories**: Categorised by type (crisis, counselling, etc.)

### Docs

Opens the documentation site ([docs.mindmeasure.co.uk](https://docs.mindmeasure.co.uk)) in a new tab with auth token passthrough. Superadmin only.

## Institutional Dashboards

Accessed from the Universities/Companies tab or via direct URL.

### University Dashboard

URL pattern: `admin.mindmeasure.co.uk/university/{slug}`

- **Wellbeing Overview**: Current wellbeing snapshot, average mood, trends
- **Engagement Metrics**: Check-in rates, participation, streak data
- **AI Summary**: Bedrock-generated insights on student wellbeing
- **Top Topics**: What students are talking about
- **Cohort Analysis**: Breakdown by year group, faculty, etc.
- **Content Hub**: Link to university-specific content management
- **Reports**: Downloadable wellbeing reports
- **Settings**: University branding, resources, users

### Company Dashboard

URL pattern: `admin.mindmeasurepro.com/company/{slug}`

Same structure as university dashboard with corporate terminology:
- Departments and offices instead of faculties
- Employees instead of students
- UK heatmap showing office locations
- Org structure management

## Content Hubs

Each institution has a content hub for managing articles and nudges. Accessed via the institutional dashboard — no standalone login.

| Hub | URL |
|---|---|
| University | [hub.mindmeasure.co.uk](https://hub.mindmeasure.co.uk) |
| Pro | [hub.mindmeasurepro.com](https://hub.mindmeasurepro.com) |

Features:
- **Article Editor**: Tiptap rich-text editor for wellbeing content
- **Nudge Manager**: Priority-based nudge cards for mobile app carousel
- **Content Pool**: Browse and adopt articles from the shared pool
- **Student/Employee Insights**: Aggregated worries and interests
