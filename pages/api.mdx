# API Documentation

## Overview

The Mind Measure API provides access to student wellbeing data and analytics.

## Authentication

All API requests require authentication using JWT tokens.

## Endpoints

### Universities
- `GET /api/universities` - List all universities
- `GET /api/universities/{id}` - Get university details

### Students  
- `GET /api/students` - List students (filtered by university)
- `GET /api/students/{id}` - Get student details

### Analytics
- `GET /api/analytics/wellbeing` - Get wellbeing analytics
- `GET /api/analytics/trends` - Get trend data
