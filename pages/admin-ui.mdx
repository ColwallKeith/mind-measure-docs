# Admin UI Guide

Complete guide to the Mind Measure administrative interface and superuser dashboard.

## Superuser Dashboard

Access the complete administrative interface at: [admin.mindmeasure.co.uk/superuser](https://admin.mindmeasure.co.uk/superuser)

### Navigation Tabs

#### Overview
- **System Statistics**: Active universities, total users, system health
- **Network Analytics**: Platform-wide wellbeing trends
- **Real-time Metrics**: Current system performance

#### Universities
- **University Management**: Direct access to all institution dashboards
- **Quick Actions**:
 - Dashboard access for Worcester, Birmingham, Manchester
 - CMS management for each institution
 - Settings and configuration

#### Security
- **Security Dashboard**: Comprehensive security monitoring
- **Compliance Status**: HIPAA, GDPR, SOC2 compliance tracking
- **Test Suites**: Access to all security testing tools
- **Audit Logs**: Real-time security event monitoring

#### Costs
- **Cost Monitoring**: Real-time AI service cost tracking
- **Usage Analytics**: Service utilization metrics
- **Budget Alerts**: Automated cost threshold notifications
- **Optimization**: Cost optimization recommendations

#### System
- **Service Health**: Real-time system status monitoring
- **Resource Metrics**: CPU, memory, storage utilization
- **Backup Management**: Automated backup monitoring
- **System Settings**: Configuration management

#### Testing
- **Security Tests**: All security validation suites
- **AWS Backend Tests**: Infrastructure testing
- **Performance Tests**: System performance validation
- **Integration Tests**: End-to-end testing
